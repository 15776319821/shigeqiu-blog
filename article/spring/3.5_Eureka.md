# Eureka


eurkeka 服务器配置

``` yml
eureka:
  client:
    registerWithEureka: false
    fetchRegistry: false
    serviceUrl:
      defaultZone: http://localhost:8761/eureka/
```

| 配置项  | 说明  |
|---|---|
| registerWithEureka=false  |  禁止向eureka注册服务，因为它自己本身就是服务器 |
|fetchRegistry=false | 这里不需要抓取注册表 |
